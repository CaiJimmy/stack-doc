# i18n

修改 `config.yaml` 的 `DefaultContentLanguage` 来选择主题语言。

```yaml
DefaultContentLanguage: en
```

目前支持的语言可以在 `i18n` 文件夹下看到。

-   `de`: 德语
-   `en`: 英语
-   `es`: 西班牙语
-   `fr`: 法语
-   `id`: 印尼语
-   `it`: 意大利语
-   `ja`: 日语
-   `ko`: 韩语
-   `nl`: 荷兰语
-   `pl`: 波兰语
-   `pt-br`: 葡萄牙语
-   `ru`: 俄语
-   `tr`: 土耳其语
-   `zh-cn`: 中文

欢迎提交翻译文件 😉
