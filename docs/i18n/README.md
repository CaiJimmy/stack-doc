# i18n

```yaml
DefaultContentLanguage: "en"
```

Set `DefaultContentLanguage` to desired language:

* `ca`: Catalan
* `de`: German
* `en`: English
* `es`: Spanish
* `fr`: French
* `id`: Indonesian
* `it`: Italian
* `ja`: Japanese
* `ko`: Korean
* `pt-br`: Portuguese
* `zh-cn`: Chinese
